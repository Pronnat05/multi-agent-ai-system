# Multi-Agent AI System

## Description
A multi-agent AI system that manages tasks, schedules, and notes using Google Cloud.

## Features
- Task management
- Notes management
- Event scheduling
- Multi-step workflow execution

## Tech Stack
- FastAPI
- Cloud Run
- Cloud SQL
- Cloud Storage

## Deployment
Backend: Cloud Run  
Frontend: Cloud Storage  

## Usage
Use API or web UI to interact with system
